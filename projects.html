DOM Play

DOM Play is an interactive JavaScript tool that highlights all dialogue lines for a selected actor using DOM traversal and data-actor attributes. It demonstrates real‑time content manipulation through event listeners, making the script easier to explore and visually follow.

Live Demo GitHub Repo

Creatures

Creatures is an interactive HTML/CSS/JavaScript project that uses data attributes to identify each creature’s type and natural environment, allowing the page to dynamically highlight them through button‑triggered DOM functions. It demonstrates how to work with nodelists, apply conditional styling, and manipulate groups of elements efficiently using JavaScript.
